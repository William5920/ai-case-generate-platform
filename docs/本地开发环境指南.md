# 本地开发环境指南

## 技术栈

| 层 | 技术 |
|----|------|
| 后端 | FastAPI + Python 3.13 |
| 数据库 | MySQL 8.0 |
| 前端 | Vue.js + Webpack |

## 环境信息

| 配置项 | 值 |
|--------|-----|
| 后端地址 | http://localhost:8000 |
| 前端地址 | http://localhost:8080 |
| MySQL 用户 | root |
| MySQL 密码 | root123456 |
| 数据库名 | ai_case_platform |

## 启动步骤

### 1. 启动 MySQL

```bash
brew services start mysql@8.0
```

验证：`mysql -u root -proot123456 -e "SELECT 1"`

### 2. 启动后端

```bash
cd backend
source venv/bin/activate
python start_server.py
```

验证：访问 http://localhost:8000，返回 `{"message":"智能测试用例平台API"}` 即正常。

### 3. 启动前端

```bash
cd frontend
npm install
npm run serve
```

## 关停步骤

1. **停止前端**：`Ctrl + C`
2. **停止后端**：`Ctrl + C`
3. **停止 MySQL**：`brew services stop mysql@8.0`

## 首次环境搭建

### 1. 安装 MySQL 8.0

```bash
brew install mysql@8.0
brew services start mysql@8.0

# 设置 root 密码
mysql -u root -e "ALTER USER 'root'@'localhost' IDENTIFIED BY 'root123456'; FLUSH PRIVILEGES;"
```

### 2. 创建数据库

```bash
mysql -u root -proot123456 -e "CREATE DATABASE IF NOT EXISTS ai_case_platform CHARACTER SET utf8mb4 COLLATE utf8mb4_0900_ai_ci;"
mysql -u root -proot123456 ai_case_platform < docs/middleware/mysql/init/init.sql
mysql -u root -proot123456 ai_case_platform < docs/middleware/mysql/init/user.sql
```

### 3. 安装 Python 依赖

```bash
cd backend
python3.13 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 4. 配置 .env

复制 `backend/.env.example` 为 `backend/.env`，参考配置：

```env
DATABASE_URL=mysql+aiomysql://root:root123456@127.0.0.1:3306/ai_case_platform?charset=utf8mb4
JWT_SECRET_KEY=ai-case-platform-secret-key-change-in-production
JWT_ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=60
REFRESH_TOKEN_EXPIRE_DAYS=7

OPENAI_API_KEY=sk-xxx
OPENAI_BASE_URL=https://api.deepseek.com
OPENAI_MODEL=deepseek-chat
OPENAI_MODEL_ANALYZE=deepseek-chat
OPENAI_MODEL_GENERATE=deepseek-chat

EMBEDDING_API_KEY=sk-xxx
EMBEDDING_BASE_URL=https://api.siliconflow.cn/v1
EMBEDDING_MODEL=BAAI/bge-large-zh-v1.5
EMBEDDING_DIM=1024
```

### 5. 配置前端代理

确保 `frontend/.env.development` 中代理指向本地后端：

```env
VUE_APP_API_URL=/api
VUE_APP_API_TARGET=http://localhost:8000
VUE_APP_BASE_URL=/
```

## 注意事项

- 知识库功能依赖 Milvus，未安装不影响核心功能使用
- 前端 dev server 通过 webpack 代理将 `/api/*` 请求转发到后端
- 数据库表结构由 SQLAlchemy 自动同步，建表 SQL 作为初始化参考
